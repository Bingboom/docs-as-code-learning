ATI — 获取模组厂商信息
==============

获取模组厂商信息，包括厂家、型号和版本。
命令格式

执行命令
^^^^^^^^

**命令：**

::

    ATI

**响应：**

::

    <CR><LF>OK<CR><LF>


参数
^^^^


-  **<manufacturer>**：
    模组厂商信息、产品名称、版本号



-  **<module_version>**：
    模组型号



-  **<soft_version>**：
    模组软件版本
